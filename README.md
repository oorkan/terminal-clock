**Description**

A simple binary clock running in your terminal.

&nbsp;

**Installation**

```bash
  chmod +x bitime.py
```

```bash
  sudo mv bitime.py /usr/bin/bitime
```

```bash
  bitime # show the clock
```

&nbsp;

**Requirements**

Requires Python>=3.8. Tested only under Debian.
